<h1>Smileschool Homepage</h1>

This repository contains the front-end code for the Smileschool homepage, a vibrant and responsive website designed to showcase "smile" tutorials, introduce instructors, highlight membership benefits, and answer frequently asked questions.

<h2>🚀 Live Demo</h2>

You can view a live preview of this project by simply opening the index.html file in your web browser.

✨ Features
Responsive Navigation: A clean and functional header with navigation links (COURSE, PRICING, LOGIN).
Hero Section: An engaging introduction with a catchy headline, keywords, and a prominent call-to-action button."Learn from the Pros" Section: Showcases expert instructors with their profiles and achievements.
Testimonial Section: Features a quote from a satisfied learner, building trust and credibility.
"Most Popular Tutorials" Section: Highlights key educational content with descriptions, instructor details, star ratings, and video durations.
"Free Membership" Benefits: Outlines the advantages of signing up for a free membership with clear icons and descriptions.
Frequently Asked Questions (F.A.Q) Section: Provides answers to common queries in an organized layout.
Social Media Footer: Includes a footer with the company logo, social media links (Facebook, Twitter, Instagram), and copyright information.

🛠️ Technologies 
UsedHTML5: For the core structure and content of the webpage.
CSS3: For styling and layout, ensuring a visually appealing and responsive design.
Font Awesome: Integrated for various icons, including stars for ratings and social media icons.

📦 Getting Started

To get a local copy of this project up and running on your machine, follow these simple steps.
Prerequisites

You only need a modern web browser (like Chrome, Firefox, Edge, or Safari) to view this project.

Installation

1. Clone the repository:
git clone https://github.com/Musabe15/alx_html_css.git

2. Navigate to the project directory:
cd alx_html_css/css_advanced

3. Open index.html:
Simply open the index.html file in your preferred web browser. The CSS (assuming it's named style.css in the same directory) will be automatically applied.

💡 Usage

This is a static website. You can navigate through its different sections by scrolling or by clicking on the header links (if implemented to scroll to sections).
